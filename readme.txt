By default, the "more" tag will add an html anchor to the link. This removes it.

== Description ==

This plugin updates the code for the "more" tag to keep it from adding html anchors to the link.

There is no formatting, widgets, shortcode, or settings. Just enable the plugin and it will change the functionaly of the built-in "more" tag.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the **xdn-more** folder to your **/wp-content/plugins/** directory or install it through the WordPress "Add New" function under the Plugins menu.
2. Activate the plugin through the 'Plugins' menu in WordPress.

> **Example Usage**
>
> `<!--more-->`
>
